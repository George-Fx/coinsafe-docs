1. Folder structure

  • coinsafe > src > assets: Contains images and other assets.
  • coinsafe > src > components: Contains reusable UI components.
  • coinsafe > src > config: Contains application configuration, including API endpoint URLs.
  • coinsafe > src > constants: Contains constants for the application.
  • coinsafe > src > hooks: Contains custom hooks.
  • coinsafe > src > items: Contains reusable item components rendered using .map
  • coinsafe > src > navigation: Contains navigation components.
  • coinsafe > src > routes: Contains application routes.
  • coinsafe > src > screens: Contains screen components.
  • coinsafe > src > store: Contains Redux store configuration.
  • coinsafe > src > text: Contains text components.
  • coinsafe > src > tokens: Contains supported tokens for the application.
  • coinsafe > src > types: Contains types for the application.
  • coinsafe > src > utils: Contains utility functions.
  • coinsafe > src > validation: Contains validation functions.

2. Install environment

Your computer should have Node.js and npm installed to manage the dependencies. You can download Node.js (which includes npm) from the following link: Node.js

3. Install dependencies

You need to install the project dependencies. To do this, open a terminal, navigate to the project directory, and run the following command: npm install

4. Pod install

To install the pods, run the following command in the project directory: cd ios && pod install

5. Run the project

To run the project, use the following command: npx react-native run-ios or npx react-native run-android

6. Publishing to App Store and Google Play

To publish your React Native application on the App Store and Google Play, you need to prepare builds for iOS and Android.

Follow the official guides for step-by-step instructions:
- For iOS: Submitting to the App Store
- For Android: Google Play Distribution

For React Native-specific instructions, refer to:
- iOS Deployment: Publishing to the App Store (React Native)
- Android Deployment: Generating a Signed APK/AAB (React Native)
